# Messaging & Push Notifications

Our Push Notification Service allows you recieve realtime notifications of events within your system. With this system, you can build robust systems that send realtime notifications to your clients on any devices they might be using.

Engage your users across multiple channels with targeted, timely, and personalized communications that drive action and build trust. This API enables you to send messages via SMS, email, and push notifications from a single integration.

### Account Setup

Visit the API keys and settings page to get your API credentials.

&#x20;Once you have your **Client ID and Client Secret, Service Keys** (from either the Staging or Production environment delivered to your email), and **MAC Key** on your api keys page,  you can start testing push notifications.
