# BVN Details LookUp

This Is a 3 Step Process Involving Retrieveing the BVN Details of A User.&#x20;

1: Generate a Unique Customer Reference for the BVN Retrieval Instance(This Tracks the BVN Retrieval Instance for that Particular BVN at that Time). \
2: Input OTP Sent to Customer Number/Email for Verification. \
3: Retrieve BVN Details.
