# mCatalyst

Welcome to the mCatalyst API documentation. This API allows you to create and manage debit instructions as well as KYC Verification and Automation for your customers, enabling seamless and automated payment processing. Whether you are looking to set up recurring payments or manage one-time transactions, this API provides the necessary endpoints to facilitate these operations efficiently.
